# Anthropic 品牌樣式

## 概覽

需要取用 Anthropic 官方品牌識別與樣式資源時，使用此技能。

**關鍵字**：品牌設計、企業識別、視覺識別、後處理、樣式設定、品牌色彩、字體排印、Anthropic 品牌、視覺格式、視覺設計

## 品牌規範

### 色彩

**主要色彩：**

- 深色：`#141413` — 主要文字與深色背景
- 淺色：`#faf9f5` — 淺色背景與深色背景上的文字
- 中灰：`#b0aea5` — 次要元素
- 淺灰：`#e8e6dc` — 輕微背景色

**強調色：**

- 橘色：`#d97757` — 主要強調色
- 藍色：`#6a9bcc` — 次要強調色
- 綠色：`#788c5d` — 第三強調色

### 字體排印

- **標題**：Poppins（備用：Arial）
- **內文**：Lora（備用：Georgia）
- **備註**：建議預先在環境中安裝字體，以獲得最佳效果

## 功能特色

### 智慧字體套用

- 標題（24pt 以上）套用 Poppins 字體
- 內文套用 Lora 字體
- 自訂字體不可用時，自動切換為 Arial/Georgia
- 在各系統上保持良好可讀性

### 文字樣式

- 標題（24pt 以上）：Poppins 字體
- 內文：Lora 字體
- 依背景色智慧選擇文字色彩
- 保留文字層級與格式

### 形狀與強調色

- 非文字形狀使用強調色
- 依序循環使用橘色、藍色、綠色強調色
- 維持視覺豐富性，同時符合品牌規範

## 技術細節

### 字體管理

- 優先使用系統已安裝的 Poppins 與 Lora 字體
- 自動備援至 Arial（標題）與 Georgia（內文）
- 無需額外安裝字體，直接使用系統現有字體
- 建議預先安裝 Poppins 與 Lora 字體以獲得最佳效果

### 色彩套用

- 使用 RGB 色彩值精準匹配品牌色
- 透過 python-pptx 的 RGBColor 類別套用
- 在不同系統上保持色彩一致性
