此技能引導創作出獨特、可上線的前端介面，避免落入「AI 廉價感」的美學陷阱。實作真實可運作的程式碼，並在視覺細節與創意選擇上精益求精。

使用者提供前端需求：一個元件、頁面、應用程式或介面。他們可能會說明用途、目標受眾或技術限制。

## 設計思維

開始編碼前，先理解情境脈絡，並確立**大膽的**美學方向：
- **目的**：這個介面解決什麼問題？誰在使用它？
- **基調**：選擇一個極端風格：極簡粗獷、最大主義混亂、復古未來主義、有機/自然、奢華/精緻、趣味/玩具感、編輯/雜誌風、野獸派/原始、裝飾藝術/幾何、柔和/粉彩、工業/實用主義等。風格選擇多元，以此為靈感，設計出忠於方向的獨特風格。
- **限制**：技術需求（框架、效能、無障礙性）。
- **差異化**：什麼讓這個介面**令人難忘**？有什麼是使用者看完之後一定會記住的？

**關鍵原則**：確立清晰的概念方向並精準執行。大膽的最大主義與精緻的極簡主義都能成功——關鍵在於有意識的抉擇，而非衝擊的強度。

接著實作可運作的程式碼（HTML/CSS/JS、React、Vue 等），需符合：
- 可上線的品質，功能完整
- 視覺上令人印象深刻且難忘
- 整體美學觀點一致連貫
- 每個細節都經過精心打磨

## 前端美學指南

重點關注：
- **字體排印**：選擇美觀、獨特且有趣的字體。避免使用 Arial、Inter 等通用字體，應選擇能提升視覺質感的獨特字體——出乎意料卻富有個性的字體選擇。搭配一款獨特的展示字體與一款精緻的內文字體。
- **色彩與主題**：確立具凝聚力的美學風格。使用 CSS 變數保持一致性。主色搭配鮮明的強調色，優於使用膽怯的均勻分散色彩。
- **動態效果**：以動畫實現特效與微互動。HTML 優先採用純 CSS 解決方案。React 專案可使用 Motion 函式庫。聚焦於高衝擊時刻：一個精心設計的頁面載入配合交錯出現（animation-delay），比散落各處的微互動更令人愉悅。善用滾動觸發與 hover 狀態帶來驚喜。
- **空間構圖**：出乎意料的版面配置。不對稱。重疊。斜線流向。打破格線的元素。大量留白或刻意的高密度排版。
- **背景與視覺細節**：創造氣氛與深度，而非預設為純色背景。加入符合整體美學的情境效果與紋理。活用漸層網格、雜訊紋理、幾何圖案、層疊透明度、戲劇性陰影、裝飾性邊框、自訂游標、顆粒覆蓋等創意形式。

**絕對禁止**使用以下通用的 AI 生成美學：過度使用的字體家族（Inter、Roboto、Arial、系統字體）、陳腐的配色方案（尤其是白色背景上的紫色漸層）、可預測的版面與元件模式，以及缺乏情境特色的千篇一律設計。

發揮創意，做出真正為當下情境而設計的意外選擇。每個設計都應是獨一無二的。在淺色與深色主題、不同字體、不同美學之間輪換。**絕對不要**重複採用相同的慣常選擇（例如 Space Grotesk）。

**重要提示**：讓實作複雜度與美學願景相匹配。最大主義設計需要大量動畫與效果的精細程式碼。極簡或精緻設計需要克制、精準，以及對間距、字排與細微細節的用心。優雅來自於對願景的精準執行。
